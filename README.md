# HVAC Data Analysis Project

This project analyzes HVAC system data using Python and Pandas.

## Files
- hvac_data.csv: Sample HVAC system data
- index.py: Python script for data analysis
- hvac_power_time.png: Power consumption over time chart

## What the project does
- Reads HVAC data from CSV
- Calculates temperature difference (delta T)
- Estimates cooling load
- Plots power consumption vs time

## Tools
- Python
- Pandas
- Matplotlib

## How to run
1. Install requirements:
pip install pandas matplotlib

2. Run:
python index.py
